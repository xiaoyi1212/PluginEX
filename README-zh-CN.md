# PluginEX

适用于SpigotMC的插件管理器

<hr>

## 语言

* [English](README.md)
* [简体中文](README-zh-CN.md)

## 适用于

* SpigotMC
* CraftBukkit
* PaperMC
* CatServer

## 命令

`/pluginex` 或 `/pex`

### load

<p>加载一个插件到服务器</p>

* name - 插件文件名.

> /pluginex load \<name>

### disable

<p>禁用一个插件</p>

* name - 插件名.
<br>

> /pluginex disable \<name>
