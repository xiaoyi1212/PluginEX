# PluginEX

Plugin manager for SpigotMC

<hr>

## Language

* [English](README.md)
* [简体中文](README-zh-CN.md)

## Server

* SpigotMC
* CraftBukkit
* PaperMC
* CatServer

## Command

`/pluginex` or `/pex`

### load

<p>Load a plugin to server.</p>

* name - Plugin file name.

> /pluginex load \<name>
